Overview

Project Summary

End-to-end quality assurance system for Sony manufacturing lines. Monitors worker actions via camera feeds, recognizes assembly procedures through a multi-model pipeline, detects errors in real time, and provides actionable feedback. Achieved 97.9% accuracy on real production data with 84% hardware cost reduction through architectural optimization, INT8 quantization, and GPU pipeline engineering.

Label Engineering

Label Redefinition

Reduced 19 color-specific action classes to 8 generic classes using Meta GNN for color splitting, targeting the same 92.3% baseline accuracy with a more generalizable label space.

Label Redefinition — Before: 19 color-specific HANARL classes (92.3%) → After: 8 generic classes + Meta GNN

Optimization

Cost Optimization

Systematic hardware cost reduction from ¥330K to ¥52.5K per station through architecture redesign, GPU pipeline optimization, INT8 quantization, and multi-instance sharing.

Cost Reduction Overview — RTX 3090 ¥330K/station → shared deployment ≤¥200K/station. TIGER+MUSE 16.8GB → TIGER+YOLOX ~5GB
Hardware Feasibility — Green/Yellow/Red rating for RTX 3090, 2080 Ti, 2070/80, 2060 across 1–4 stations
Architecture Transformation — Remove MUSE, keep only YOLOX (80% reduction in GPU ops, memory, kernels)
4-Pronged Strategy — S1: Architecture Redesign, S2: GPU Pipeline Optimization, S3: INT8 Quantization, S4: Multi-Instance Architecture
