Overview

Seamless Virtual-Real Object Occlusion

Real-time hand-mesh generation enabling seamless virtual-real object occlusion in AR pass-through mode. Engineered custom triangularization achieving 88.5x speedup over baseline Open3D methods, allowing convincing hand-over-virtual-object rendering at interactive frame rates on constrained hardware.
